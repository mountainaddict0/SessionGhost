**Quiz #01: Convert Moore Machine to Mealy Machine**

Moore Machine (from image):
- States: q0 (output=0), q1 (output=1), q2 (output=0)
- Transitions:
  - δ(q0, 1) = q1
  - δ(q1, 1) = q0
  - δ(q1, 0) = q2
  - δ(q2, 1) = q1
  - δ(q2, 0) = q0

Mealy Machine conversion rule: λ_mealy(q, a) = λ_moore(δ(q, a))
(Output on transition = output of destination state in Moore)

Mealy transitions:
- (q0, 1) → q1, output = λ_moore(q1) = 1
- (q1, 1) → q0, output = λ_moore(q0) = 0
- (q1, 0) → q2, output = λ_moore(q2) = 0
- (q2, 1) → q1, output = λ_moore(q1) = 1
- (q2, 0) → q0, output = λ_moore(q0) = 0

So Mealy transitions (state, input/output → next state):
- q0 --1/1→ q1
- q1 --1/0→ q0
- q1 --0/0→ q2
- q2 --1/1→ q1
- q2 --0/0→ q0

Note: q0 has no 0-input transition shown in the Moore machine diagram, so I won't add one.
